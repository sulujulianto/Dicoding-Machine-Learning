# Dicoding Machine Learning Engineer

Learning Path **Machine Learning Engineer** ini didapatkan dari program beasiswa [DBS Foundation](https://www.dbs.com/spark/index/id_id/site/codingcamp/index.html).

## Learning Path

- [x] [Memulai Pemrograman Dengan Python](https://www.dicoding.com/academies/86)  
  Belajar Python yang menjadi landasan penting berbagai tren industri seperti ilmu data, pembelajaran mesin, dan manajemen infrastruktur.

- [x] [Belajar Dasar Visualisasi Data](https://www.dicoding.com/academies/177)  
  Pelajari teknik dasar untuk representasi hasil data secara visual sehingga dapat mengkomunikasikan output dari data yang telah terproses.

- [x] [Belajar Machine Learning untuk Pemula](https://www.dicoding.com/academies/184)  
  Pelajari materi dasar pengembangan machine learning dan langkah menciptakan model machine learning pertamamu untuk memproses data.

- [x] [Belajar Fundamental Deep Learning](https://www.dicoding.com/academies/185/corridor)  
  Kelas lanjutan untuk *Machine Learning Engineer* yang mempelajari teknik *Deep Learning* menggunakan TensorFlow dan Keras, mencakup topik *Computer Vision*, *Natural Language Processing*, dan *Reinforcement Learning*.
